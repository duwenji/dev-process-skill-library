# Contributing

改善提案や Skill 追加は歓迎します。

## 基本方針
- 再利用しやすいプロセス知識を優先してください。
- 具体的な利用トリガーと成果物を明記してください。
- 大きな変更は `Issue` で背景を共有してから進めてください。

## 推奨フロー
1. `Issue` を作成する
2. branch を切る
3. `skills/` / `templates/` / `governance/` を更新する
4. `README.md` と関連資料を更新する
5. `VALIDATION_CHECKLIST.md` を確認する
6. `Pull Request` を作成する
