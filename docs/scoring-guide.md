# 優先度採点ガイド

各 Skill を5観点で 1〜5 点評価し、合計点で優先度を判定します。

## 採点観点

| 観点 | 説明 |
|---|---|
| 影響度 | 品質、納期、顧客影響への寄与 |
| 発生頻度 | 対象課題が発生する頻度 |
| 緊急度 | 早期導入の必要性 |
| 実装容易性 | 導入工数の小ささ |
| 依存関係 | 他 Skill の前提となる度合い |

## 判定基準

- High: 20点以上
- Medium: 14〜19点
- Low: 13点以下

## 採点ルール

- 採点者: Owner と Reviewer（PM + QA）の合議
- 採点タイミング: 新規登録時、四半期レビュー時、重大インシデント発生後
- 同点時の優先: 依存関係 > 影響度 > 緊急度
