# Sub-Skills テンプレート ガイド

<!-- markdownlint-disable MD022 MD024 MD025 MD031 MD032 MD040 MD053 MD058 MD060 -->

各 Skill 配下の `sub-skills/` ディレクトリに配置する Phase ごとの詳細手順ファイルの構造を定義します。

---

## ファイル構成

```
skills/[カテゴリ]/[スキル名]/sub-skills/
  ├─ phase1-[KEY].md      # Phase 1 詳細手順（KEY: discovery, investigation, planning, preparation etc）
  ├─ phase2-[KEY].md      # Phase 2 詳細手順（KEY: implementation, design-implementation, execution-planning etc）
  ├─ phase3-[KEY].md      # Phase 3 詳細手順（KEY: testing, validation, feedback-and-adjustment etc）
  └─ phase4-reporting.md  # Phase 4 詳細手順（名前固定）
```

---

## 各 Phase の標準セクション構成

### Phase 1 テンプレート（phase1-[KEY].md）

- [ ] 条件A: Phase 1 の出力が十分な品質か

## 共通セクション（全 Phase で推奨）

### 確認チェック
各段階終了時に、「次へ進むための確認」セクションで最低限の確認項目を定義します。

### 出力物リスト
各段階で何を出力すべきかを明示します。

### 次ステップ
各段階完了後、次のステップを明確に指示します。

---

## Customization Hints

- **phase1-[KEY].md** の KEY: discovery, investigation, assessment, planning, preparation, refinement 等から選流
- **phase2-[KEY].md** の KEY: implementation, design-implementation, execution-planning 等から選択
- **phase3-[KEY].md** の KEY: validation, testing, feedback-and-adjustment 等から選択
- **phase4-reporting.md** は全スキル共通で固定
