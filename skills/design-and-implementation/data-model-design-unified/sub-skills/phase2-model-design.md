# Phase 2: 論理/物理モデル決定

## 目的
採用したモデル案を、実装可能な論理モデル・物理モデルへ具体化する。

## 実施ポイント
1. テーブル、列、型、NULL可否、制約を定義する
2. PK/FK、インデックス、命名規則を統一する
3. データ辞書を作成し、要件IDと対応付ける
4. DDL 方針（新規、変更、廃止）を明文化する
