# Phase 2 Sub-Skill: 設計決定・実装（段階7-9）

## 概要
Phase 2では、開発者が方式案を決定し、AIが決定案を実装、開発者がレビューします。

## ゲート条件 #1（段階7）
- [ ] 方式案比較が成立している
- [ ] 受入条件との適合が説明されている
- [ ] 主要リスクと緩和策が示されている

## 段階8の実施ポイント
1. 選定方式に沿って最小差分で実装
2. 変更理由をコード差分に紐付け
3. ビルドと静的チェックを実施

## 段階9のレビュー観点
- 選定方式との整合
- 可読性、保守性、例外処理
- 既存機能への副作用

## 出力
- 変更ファイル一覧
- 差分サマリ（before/after）
- 実装根拠
