# 調査チェックリスト（Investigation Checklist）

本チェックリストは、Phase 1「調査・分析・対応案決定」（段階1-6）で使用する検査項目テンプレートです。

## 利用方法
- **対象**: 段階4「AI が原因調査を実施」で、調査漏れを防ぐ
- **出力形式**: Markdown チェックリスト
- **記録方法**: AI改善/ ログファイルに記入
